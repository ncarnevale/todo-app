# Fullstack Todo App

This repo contains a simple fullstack To-Do app built with:

- **Frontend:** React, TypeScript, Vite, React Query
- **Backend:** ASP.NET Core, EF Core, SQLite

## Project Structure

- `todo-frontend/` — React frontend
- `todo-backend/` — ASP.NET Core API + integration tests

## Getting Started

Each project contains its own README with setup and run instructions.
